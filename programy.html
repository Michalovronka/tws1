Programy
Existuje řada softwarových nástrojů určených pro 3D tisk, které poskytují uživatelům široké možnosti pro vytváření, úpravy a přípravu 3D modelů. Mezi nejpopulárnější softwary patří například Autodesk Fusion 360, který kombinuje funkce CAD a CAM (Computer-Aided Manufacturing). Tento software umožňuje uživatelům vytvářet složité 3D modely, provádět analýzy pevnosti a optimalizovat geometrii pro 3D tisk. Další známý software je Blender, který je více zaměřen na tvorbu polygonálních modelů a animaci. Má širokou paletu nástrojů pro modelování, texturování a renderování, a také obsahuje rozšíření pro generování podporových struktur pro 3D tisk.
Slicing software
Slicing software je dalším důležitým typem softwaru pro 3D tisk. Jedním z nejpopulárnějších slicing software je Ultimaker Cura, který je kompatibilní s řadou 3D tiskáren. Tento software analyzuje 3D model a generuje instrukce pro 3D tiskárnu, jako je tloušťka vrstvy, rychlost tisku a podpůrné struktury. Uživatelé mohou také upravovat různé tiskové parametry podle svých potřeb. Další slicing software zahrnují například PrusaSlicer, Simplify3D nebo Slic3r. Tyto softwary poskytují pokročilé možnosti pro přizpůsobení tisku, vrstev a podpůrných struktur a umožňují dosáhnout vyšší kvality a přesnosti tisknutých modelů.
